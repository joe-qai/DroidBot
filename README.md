# DroidBot

AI-Powered Android Compatibility Testing Platform

## Overview

DroidBot is an AI-driven automated testing platform for Android application compatibility testing. With natural language processing capabilities, it enables users to describe test requirements in plain language while the AI Agent automatically plans tasks, generates executable scripts, and executes tests across multiple devices in parallel.

## Features

- **AI Script Generation**: Describe test requirements in natural language, and AI automatically generates executable test scripts
- **Multi-Device Parallel Execution**: Connect multiple Android devices simultaneously for compatibility testing
- **Intelligent Reports**: Auto-generated detailed test reports with AI-powered failure analysis
- **APK Management**: Easy APK file upload and management with version control
- **Scheduled Tasks**: Configure automated test execution with CI/CD integration support
- **Device Management**: Real-time device status monitoring with automatic reconnection

## Quick Start

1. Open `index.html` in a browser to view the project introduction
2. Click "体验 Demo" to access the interactive demo dashboard
3. Configure your LLM settings in "LLM配置"
4. Connect Android devices in "设备管理"
5. Use AI Script Generation to create tests with natural language

## Tech Stack

- Frontend: HTML5, CSS3, JavaScript
- Backend: Python (uiautomator2, FastAPI)
- AI: LLM integration (OpenAI, Anthropic, Local Models)
